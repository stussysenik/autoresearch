### Quick Formula

```
[Action Verb] + [specific file/component] + [what to change] + [expected outcome]

Example:
  "Fix the null pointer in src/auth/login.ts:42 — getUser() returns
   undefined when token is expired. Should redirect to /login instead."
```

### Optimal Prompt Structure

1. **Start** with a Tier 1 action verb (fix, implement, add, create, update)
2. **Name** the file path and function/component
3. **Describe** expected vs actual behavior
4. **Include** error output or code block if debugging
5. **Use** numbered steps for multi-part requests
6. **Keep** to 15-100 words (sweet spot range)
7. **End** sessions with "commit" or "run tests"
